> 現行の実装はダーク系の**旧テーマ**（新規流用禁止・移行予定）。

## Development

When starting the dev server, use background mode:

```
astro dev --background
```

Manage the background server with `astro dev stop`, `astro dev status`, and `astro dev logs`.
